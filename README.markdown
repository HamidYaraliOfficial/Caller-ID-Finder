# 🇬🇧 English

## Overview

Caller ID Finder is a lightweight web application that allows users to search and identify phone number ownership information through an external API. The interface supports Persian, English, and Chinese languages and provides a modern responsive UI with smooth animations and multilingual support.

### Features

- Responsive modern UI
- Persian, English, and Chinese language support
- Phone number lookup
- API integration
- Animated loading screen
- Mobile-friendly design
- Lightweight Python server
- CORS enabled server

---

## Project Structure

```text
index.html     Frontend application
server.py      Python HTTP server
```

---

## Requirements

- Python 3.8+
- Modern web browser
- Internet connection for API requests

---

## Install Required Libraries

This project mainly uses Python built-in libraries.

If you want to install common dependencies:

```bash
pip install --upgrade pip
```

Optional:

```bash
pip install requests
```

---

## Installation

### Windows

```bash
python server.py
```

### Linux

```bash
python3 server.py
```

### Custom Port

```bash
python server.py 8080
```

---

## Run Application

After starting the server, open:

```text
http://localhost:85
```

or your selected port.

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Python
- HTTPServer

---

## Security Notice

Before production deployment:

- Move API tokens to environment variables
- Use HTTPS
- Restrict API access
- Enable rate limiting
